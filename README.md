# RS-31_Haponko_Nikita_RTOS_LR5

**Номер групи:** РС-31

**Автор:** Гапонько Нікіта

**ОПЕРАЦІЙНІ СИСТЕМИ РЕАЛЬНОГО ЧАСУ ТА 32-БІТНІ МК**
**Лабораторна робота 5.** Технологія використання аналого-цифрового перетворювача (АЦП) при розробці ПЗ ОСРЧ.

**Обладнання:** STM32F411 Discovery, CubeMX, Keil.

**Мета:** Ознайомитися з принципами роботи АЦП у мікроконтролерах STM32, навчитися налаштовувати канали оцифровки та перетворювати отримані дані у фізичні величини (напругу).

__________________________________________________________________________________________

**Завдання до лабораторної роботи (Варіант 4):**
1. Зчитати сигнал з каналу **PA4** за допомогою АЦП.
2. Реалізувати перетворення отриманого цифрового значення у напругу.
3. Якщо напруга **більше 3.0 В** — увімкнути світлодіод **PD12**, інакше — вимкнути його.

Цей проєкт демонструє роботу з аналоговими сигналами та базову логіку прийняття рішень на основі виміряних значень у реальному часі.

### 👨🏼‍💻 Функціонал:
* **ADC Data Acquisition**: Циклічне опитування каналу АЦП для отримання 12-бітного значення сигналу.
* **Voltage Conversion**: Програмний перерахунок «сирих» даних АЦП у вольти (від 0 до 3.3 В).
* **Threshold Control**: Автоматичне керування світлодіодом PD12 залежить від заданого порогу напруги (3.0 В).

### 📋 Ключові моменти реалізації:
1. **ADC Configuration**: Налаштування піна PA4 в режим аналогового входу та конфігурація ADC1.
2. **Polling Mode**: Використання функції `HAL_ADC_PollForConversion` для очікування завершення вимірювання.
3. **Formula**: Розрахунок напруги за формулою: $U = (Value \times 3.3) / 4095$.
